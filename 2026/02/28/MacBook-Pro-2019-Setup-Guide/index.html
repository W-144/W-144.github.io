MacBook-Pro-2019-Setup-Guide

注意 :本篇适用于抹掉mac完全重装Windows如果只是想简单的装个双系统完全没必要这么麻烦

提醒：在执行所有操作之前请确定在Mac电脑上已经没有重要的文件

Part 1. 准备工具

  • Windows 10/11 镜像 (ISO 文件)
  • 至少 32GB 的 U 盘
  • 稳定的网络连接 (用于下载驱动)
  • 外接有线键盘（重要，必要）
  • 外接鼠标（非必要，但是有鼠标操作会简单很多）
  • 另外一个Windows电脑
  • 扩展坞（把键盘接入macbook用）

    重要：如果你没有确定的完好的外接键盘一定不要轻易尝试本流程，在装好系统后因为没有驱动，用不了触控板容易卡死在windows界面。请确保有所有的准备工具再进行尝试。

Part 2. 核心步骤

1.镜像安装

从一个靠谱的渠道安装Windows镜像即可，将其拷贝到准备的U盘之中。

2.制作启动盘

去Rufus官网 https://rufus.ie/zh/ 下载Rufus工具，推荐x64便携版

⚙️ 针对 2019 款 Mac 的关键设置（重要！）

在你制作启动盘时，请务必按照以下截图般的逻辑进行设置，否则 Mac 可能无法识别：

  1. 设备：选择你插入的 U 盘（注意：U 盘会被格式化，请备份资料）。

  2. 引导类型选择：选择你下载好的 Windows 10 或 11 的 ISO 镜像。

  3. 分区方案：**务必选择 GPT**（2019 款 Mac 使用 UEFI 引导，必须选 GPT）。

  4. 目标系统类型：选择 UEFI (非 CSM)。

  5. 文件系统：**务必选择 NTFS**（这是解决“文件过大”报错的核心步骤）。

    注意：在 Rufus 制作时，请确保勾选了以下几项以避免卡住：

  • Remove requirement for an online Microsoft account（移除在线账户要求）。

  • **Disable data collection (Skip privacy questions)**（跳过隐私设置和联网问答）。

  • Create a local account（创建本地账户）。

3. 驱动安装

注意：Intel 芯片的驱动通常需要通过 Boot Camp 助理下载。

在BootCamp界面点击macbook左上角的mac上自己状态栏里的点击- “操作” (Action) -> “下载 Windows 支持软件”。那么就可以得到一个名为Windows support的软件，将这个文件放到U盘的根目录下。

提醒：一定要先用Rufus制作完启动盘再把驱动拖到U盘根目录下。

4.关闭安全模式

由于2019款Mac采用了T2安全芯片，所以一定要有这个步骤

如果不做这一步，你的 Mac 将禁止从 U 盘启动，也无法识别 Windows 安装程序对硬盘的改动。

  1. 关机并重启，立即长按 Command (⌘) + R 进入恢复模式。

  2. 在上方菜单栏点击 “实用工具” -> “启动安全性实用工具”。

  3. 选择 “无安全性”。

  4. 在下方选择 “允许从外部介质启动”。

5.进入windows启动盘

  • 插入制作好的 Windows 安装 U 盘。

  • 重启 Mac，立即长按 Option (⌥) 键。

  • 选择黄色图标的 EFI Boot。
    可能用得到的冷知识（强制关机 Mac：长按 Touch ID 键（那个黑方块）10 秒以上直到黑屏。）

6.Windows安装界面

进入EFI Boot后你会发现触控板和mac自己的键盘都用不了了，这个时候就需要外接键盘，将外接键盘接入电脑。
Windows 安装程序完全可以用键盘操作。请记住这几个快捷键：

  • Tab 键：在不同的选项、按钮之间切换（你会看到虚线框在移动）。

  • **空格键 (Space)**：勾选/取消勾选（比如“我接受许可条款”）。

  • **方向键 (↑ ↓ ← →)**：在列表里选择分区或语言。

  • Enter 键：点击“下一步”或“确定”。
    到了选择安装Windows11安装位置的界面，会发现磁盘0分区1，磁盘0分区2等分区，在这里可以把macOS对应的分区格式化：

🛠️ 分区“三部曲”：删、选、下一位

  1. 选中最大的分区：

    • 按 Tab 键，直到光标（虚线框）跳到分区列表里。

    • 使用 ↓ 方向键，选中那个容量最大的分区（通常是数百 GB，那是你刚才为 macOS 建立的“家”）。

  2. 执行“强拆”（删除）：

    • 按 **Alt + D**（删除的快捷键）。

    • 系统会弹出一个警告，问你是否确定删除数据。按 **Enter**（确定）。

    • 注意：如果你看到好几个小分区（比如几百 MB 的），建议把它们全都删掉，直到列表里只剩下一个：“驱动器 0 未分配的空间”。

  3. 激活安装：

    • 确保选中的是那个整块的 “未分配的空间”。

    • 直接按 **Alt + N**（下一步的快捷键），或者按 Tab 切换到“下一步”按钮并按 **Enter**。

7.安装驱动

OK，经过以上步骤，进入windows界面就说明windows操作系统已经安装好了，接下来就可以安装驱动了：

1. 找到驱动文件夹

  • 快捷键：按下键盘上的 Win + E 打开“文件资源管理器”。

  • 导航：按 Tab 键或 方向键 选中左侧的 **“此电脑” (This PC)**。

  • 找到 U 盘：在右侧找到你的安装 U 盘，按 Enter 进去。

  • 目标文件夹：寻找名为 BootCamp 或 WindowsSupport 的文件夹。（注意，这里记得不要在U盘图标对应的文件夹下面找，经过Rufus的制作，U盘里面你放进去文件的储存路径对应的盘符的图标已经变成了Windows图标）

2. 运行安装程序

  1. 进入该文件夹后，找到一个叫 Setup (或者 Setup.exe) 的文件，它的图标通常是一个灰色的正方形。

  2. 选中它，按 Enter 运行。

  3. 用户账户控制：弹窗问你是否允许更改时，按 Alt + Y (代表“是”)。

3. 安装过程中的交互

  • 欢迎界面：直接按 Enter (下一步)。

  • 许可协议：按 Tab 选中“我接受”，然后按 **Enter**。

  • 静候佳音：安装程序会开始自动刷入显卡、声卡、T2 芯片和最重要的 Touch Bar 驱动。

  • 屏幕闪烁：期间屏幕可能会黑几秒，这是在装显卡驱动，属于正常现象。

4. 重启“复活”硬件

  • 安装完成后，会提示你 **“重新启动” (Restart)**。

  • 务必点击重启。

  • 奇迹时刻：重启进入 Windows 后，你会发现触控板有了压力反馈、Touch Bar 亮了起来、扬声器也有了声音。

有的时候你会发现setup程序进度条一直是空白（百分之零）没动，打开任务管理器发现setup程序占用的内存和cpu都是0。
当然，这种情况也是非常正常的，这里分享一种解决方案，除此之外也可以尝试一下结束任务再重新安装，或者重启之后再重新安装，如果没有解决成功，就请大家自己查阅资料找找其他办法。

下面是一种解决方法

  • 按 Win + E 打开文件资源管理器。

  • 进入 U盘 > BootCamp > Drivers > Apple。

  • 在当前文件夹中，利用方向键向下找，看到那个图标是小盒子的 BootCamp (Windows Installer 类型，大小约 4,656 KB)。

  • 如果这里运行这个因为没有启动转换器运行不了，可以尝试以下方法：

  • 1. 以管理员身份打开命令行 (CMD)

  • 按下 Win + S 键，输入 **cmd**。

  • 看到“命令提示符”后，不要直接回车。按下 **Ctrl + Shift + Enter**。

  • 如果弹出“用户账户控制”窗口，按下 Alt + Y 点击“是”。

2. 定位到驱动文件夹

在黑色的窗口里，输入以下命令（假设你的 U 盘盘符是 F，如果不是请替换）：

F: （回车） cd BootCamp\Drivers\Apple （回车）（这里记得把路径替换成你的Apple文件夹对应的地址）

3. 使用 msiexec 命令强行运行

输入以下命令来安装最关键的触控板驱动：

msiexec /i BootCamp.msi （回车）

  • 如果弹出了安装窗口：直接一路回车（Next）到底。

  • 如果报错“必须通过启动转换助理”： 尝试这个“越狱”命令：

    msiexec /i BootCamp.msi /qn （回车）

如果运行了之后触控板还是没好，但是没报错尝试再次打开最开始的那个Setup文件夹，尝试重新安装，说不定这次就可以直接下下来了

8.更新，升级驱动

虽然 Setup.exe 装好了，但为了系统长久稳定，建议再做两件事：

  • 运行 Apple Software Update： 在开始菜单里搜索这个程序。它会检查有没有针对 T2 芯片或显卡的最新更新（通常会有针对 Windows 11 的优化补丁）。
    • 检查 Boot Camp 控制面板： 右下角托盘里应该有一个灰色的小菱形图标。点开它可以设置触控板的点击行为（比如“轻点以点击”）以及默认的启动系统。

9.检查更新

如果你已经完成了上述所有的步骤，那么恭喜你，现在你的Mac电脑重装Windows已经基本完成了接下来你可以打开设置然后检查更新，然后有可能会发现里面还有一些驱动的包还是没有下好并且显示的是百分之零，这个时候重启电脑然后再次进入这个界面点击一下安装，就可以安装了。
最后把所有驱动都装好了之后就可以更新Windows了。

Part 3. 激活与后续

  • MAS 激活工具
    由于mac的架构还是和传统的windows有所不同所以有的时候用激活软件还是会出现“NullReferenceException”的报错，这里分享一种解决方法，可以激活Windows和office

🚀 MAS 一键激活步骤

请在网络连通的状态下，按照以下键盘操作执行：

  1. 打开管理员 PowerShell：

    • 按下 Win + X 键，在弹出的菜单中选择 “终端(管理员)” 或 “Windows PowerShell (管理员)”。

    • 如果弹出权限确认窗口，按下 Alt + Y 点击“是”。

  2. 输入指令：

    • 在蓝色的窗口中，直接输入以下代码（建议手动敲一遍，以免复制不全）：

      irm https://get.activated.win | iex

    • 按下 **Enter**。

  3. 选择模式：

    • 稍等片刻，会弹出一个蓝底黄字的菜单。

    • 按下数字键 1 (选择 HWID 模式)。

    • HWID 是“硬件 ID 激活”，它会直接在微软服务器上给你的这台 2019 款 MBP 挂上永久正版授权，以后重装系统也会自动激活。

🛠️ 容易遗漏的“补漏神器” (避坑必看)

重装完系统并不代表结束，为了让这台 2019 款 MBP 跑得更“稳”，你还需要打上这几个技术补丁。

1. 解决时间不同步 (时差 8 小时)

痛点：你会发现 Windows 的时间永远比北京时间慢（或快）8 小时。这是因为 Mac 把主板时间看作 **UTC (世界标准时)**，而 Windows 默认看作 **RTC (本地时间)**。

方案：不需要手动调时间，直接给注册表来一刀，让 Windows 也认 UTC 逻辑。

  • 操作：以管理员身份打开 CMD，复制粘贴以下命令并按回车：

    DOS

  • 效果：重启后，时间从此稳如老狗，再也不会反复横跳了。

2. 屏幕缩放 (拯救视力)

痛点：2019 款 MBP 的视网膜屏幕分辨率极高，默认 100% 缩放下的图标和字体小到需要放大镜。

方案：手动调整 DPI 缩放。

  • 操作：进入 设置 -> 系统 -> 屏幕 -> 缩放与布局。

  • 建议：

    • **150%**：适合追求显示内容多的同学。

    • **200% (推荐)**：最接近 macOS 的 Retina 质感，看着最舒服。

3. 压制 Intel CPU 的“怒火” (Macs Fan Control)

痛点：Windows 下的苹果驱动对风扇调度非常消极，经常 CPU 都 80°C 了风扇才慢悠悠转动，导致键盘烫手甚至降频卡顿。

方案：下载 **Macs Fan Control (Windows 版)**。

  • 核心设置：

    • 不要用默认的“自动”。

    • 选择 “基于传感器值控制” -> 选择 “CPU PECI”。

    • 设置范围：建议设定为 45°C 开始加速，85°C 达到满速。

  • 效果：虽然风扇声会大一点，但能保住你的 Intel i7/i9 不因过热而“罢工”。
